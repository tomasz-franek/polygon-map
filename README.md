# Drawing polygons on Map

### Requirements:

- MongoDB backend database
- Angular 19+
- Java 21

### Used components

- Map [OpenStreetMap:]
- Library leaflet
- Bootstrap

### Start backend

```bash
cd backend
mvn spring-boot:run
```

### Start frontend (viewer)

```bash
cd viewer
npm install
run start
```
